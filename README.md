## Taskig
Taskig is a task managment CLI tool that integreats with code comments for a seamless intergration directly with the source aswell as a shallow learning curve.
Taskig is planned to integreat with github trough a bot that will be able to open/close and refrence issues from code comments aswell as refrence code from issues.
